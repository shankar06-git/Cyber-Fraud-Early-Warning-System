# Cyber Fraud Early Warning System

An AI-powered phishing and cyber fraud detection system.

## Features
- Detects phishing links using Machine Learning
- Integrates VirusTotal API for threat intelligence
- Real-time fraud detection
- Early warning system for suspicious URLs

## Technologies Used
- Python
- Machine Learning
- Scikit-learn
- VirusTotal API
- Flask / Streamlit (if you used it)

## Project Structure
app.py – Main application  
virustotal_checker.py – VirusTotal API integration  
model.pkl – Trained ML model  
vectorizer.pkl – Text vectorizer for prediction  

## Future Improvements
- Browser extension for phishing detection
- Real-time threat database
- Dashboard for cyber alerts
